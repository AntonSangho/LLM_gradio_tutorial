# ollama와 gradio를 활용하여 웹 앱 만들기

## 설치
1. ollama 설치
```
pip install ollama  
```
2. openai 설치
```
pip install openai
```

## 실행
1. 터미널에서 ollama serve를 실행 
2. http://localhost:11434에서 "Ollama is running" 확인
